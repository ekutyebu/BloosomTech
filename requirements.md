
# Requirement Analysis: BloomTech Internship Program

## 1. Project Overview
**BloomTech** is a specialized internship portal designed to facilitate a high-tier learning experience for aspiring developers and designers. 

## 2. Lead Project Architects
*   **Ekuty Ebu:** Lead Software Architect (Backend & Cloud Infrastructure)
*   **Mbakwa Blessing:** Lead UI/UX Strategist (Design Systems & User Experience)

## 3. Core Functional Requirements
1.  **Immersive Track Selection:** Interactive UI to explore Frontend, Design, and Backend curricula.
2.  **Engineering Spotlight:** Dedicated section to showcase the project's lead engineers to establish credibility.
3.  **Animated User Journey:** Fluid transitions (Framer Motion) to maintain a premium feel.
4.  **Responsive Design:** Flawless execution across mobile, tablet, and ultra-wide screens.

## 4. Visual Identity
*   **Aesthetic:** "Modern Professional Grace"
*   **Target Palette:** Blossom Pink, Rose Gold, Deep Charcoal.
*   **Typography:** Playfair Display (Serif) for headings; Plus Jakarta Sans (Sans-Serif) for UI.
