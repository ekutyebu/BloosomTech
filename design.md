
# System Design: BloomTech Internship Program

## 1. Landing Page Structure
*   **Hero Section:** High-impact visual area with the program title, mission statement, and primary CTA.
*   **About Section:** Deep dive into the "BloomTech" mission and what sets this internship apart.
*   **Programs / Services Section:** Detailed cards showcasing different tracks (Frontend, UI/UX, Backend).
*   **Call-To-Action (CTA) Section:** A final nudge for prospective interns to register.
*   **Footer:** Essential contact info, copyright, and social links.

## 2. Page Layout
The page follows a modern **Single Page Application (SPA)** architecture with a vertical "F-Pattern" layout. Each section is clearly demarcated by subtle background color shifts (Soft Pink to White) to maintain visual hierarchy.

## 3. Color Scheme
*   **Primary:** Soft Blossom Pink (`#FDF2F8`) - Used for backgrounds.
*   **Secondary:** Rose Gold Accents (`#FB7185`) - Used for buttons and highlights.
*   **Typography:** Charcoal Gray (`#374151`) - For high readability.
*   **Accent:** Deep Magenta (`#BE185D`) - For primary headings and emphasized text.

## 4. Navigation Flow
1.  User arrives at the **Hero** section.
2.  Scrolls to **About** to understand the program's values.
3.  Explores specific **Programs** to find a fit.
4.  Views **Benefits** to confirm the value.
5.  Clicks **Register** to complete the journey.
